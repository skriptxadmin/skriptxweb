Day 2: SCSS Basics
Objective: Learn the fundamentals of Bootstrap and SCSS.
Morning Session:
  • CSS Frameworks Overview:
    • Introduction to Bootstrap and its components.
    • Setting up Bootstrap in a project.
  • SCSS Basics:
    • Introduction to SCSS syntax and features.
    • Variables, nesting, and partials.
Afternoon Session:
  • Combining Bootstrap and SCSS:
    • Integrating Bootstrap with custom SCSS.
    • Creating a basic layout using Bootstrap components and SCSS customization.
  • Hands-On Practice:
    • Building a simple web page using Bootstrap and SCSS.
    • Customizing Bootstrap variables (e.g., colors, fonts).
