Day 1: Introduction and Setup
Objective: Familiarize interns with the project, tools, and team.
Morning Session:
  • Welcome and Orientation:
    • Introductions to team and project.
    • Overview of company culture and expectations.
    • Tour of the workplace (or virtual meet-and-greet for remote interns).
  • Project Overview:
    • Introduction to the project scope and objectives.
    • Review of project timeline and key milestones.
Afternoon Session:
  • Technical Setup:
    • Setting up development environments.
    • Installation of necessary tools and software (e.g., IDE, version control systems).
  • Introduction to Git and Version Control:
    • Basic Git commands and workflow.
    • Creating branches, committing changes, and merging.
